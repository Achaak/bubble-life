# Weight Vital

## setWeight()

Set the weight of the Bubble.

```ts
import { setWeight } from "@bubble/store";

setWeight({
  value: 42,
});
```

#### Parameters

| Parameter | Description              | Type     | Default    |
| --------- | ------------------------ | -------- | ---------- |
| value     | The weight of the Bubble | `number` | `required` |

---

## resetWeight()

Reset the weight of the Bubble.

```ts
import { resetWeight } from "@bubble/store";

resetWeight();
```

---

## addWeight()

Add to the weight of the Bubble.

```ts
import { addWeight } from "@bubble/store";

  value: 42,
});
```

#### Parameters

| Parameter | Description              | Type     | Default    |
| --------- | ------------------------ | -------- | ---------- |
| value     | The weight of the Bubble | `number` | `required` |
