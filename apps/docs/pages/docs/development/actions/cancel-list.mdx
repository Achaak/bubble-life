# Cancel List Actions

## hasActionInCancelListByName()

Get if the given action is in the cancel list by name.

### Use in app module

```typescript
import { hasActionInCancelListByName } from '@bubble/store'

const bool = hasActionInCancelListByName({
  name: 'ActionName',
})
```

### Use in controller module

```typescript
import { socket } from '@bubble/common'

const socket = socket()
socket.emit('hasActionInCancelListByName', {
  name: 'ActionName',
})
```

#### Parameters

| Parameter | Description                      | Type     | Default    |
| --------- | -------------------------------- | -------- | ---------- |
| name      | Name of the action to check for. | `string` | `required` |

#### Return value

```
boolean
```

## addActionInCancelList()

<Callout emoji="⚠️">
  This function must follow the nomenclature of actions available in the core package.
</Callout>

Add an action to the cancel list.

### Use in app module

```typescript
import { addActionInCancelList } from "@bubble/store";

addActionInCancelList({
  id: "ActionId"
  name: "eat",
  start: new Date().getTime(),
  duration: 5000,
  startFunction: "eat:start",
  updateFunction: "eat:update",
  endFunction: "eat:end",
  cancelFunction: "eat:cancel",
  importance: 2,
  elements: {
    onomatopoeia: {
      name: "eat",
    },
    ...
  },
  animation: 'bounce',
  memory: {
    startAt: new Date().getTime(),
    ...
  },
});
```

### Use in controller module

```typescript
import { socket } from '@bubble/common'

const socket = socket()
socket.emit('addActionInCancelList', {
  id: "ActionId"
  name: "eat",
  start: new Date().getTime(),
  duration: 5000,
  startFunction: "eat:start",
  updateFunction: "eat:update",
  endFunction: "eat:end",
  cancelFunction: "eat:cancel",
  importance: 2,
  elements: {
    onomatopoeia: {
      name: "eat",
    },
    ...
  },
  animation: 'bounce',
  memory: {
    startAt: new Date().getTime(),
    ...
  },
})
```

#### Parameters

| Parameter      | Description                                         | Type              | Default     |
| -------------- | --------------------------------------------------- | ----------------- | ----------- |
| id             | ID of the action to add to the cancel list.         | `string`          | `undefined` |
| name           | Name of the action to add.                          | `string`          | `required`  |
| start          | Start time of the action.                           | `number`          | `required`  |
| duration       | Duration of the action.                             | `number`          | `required`  |
| startFunction  | Function name to call when the action starts.       | `string`          | `undefined` |
| updateFunction | Function name to call when the action updates.      | `string`          | `undefined` |
| endFunction    | Function name to call when the action ends.         | `string`          | `undefined` |
| cancelFunction | Function name to call when the action is cancelled. | `string`          | `undefined` |
| importance     | Importance of the action.                           | `1, 2 or 3`       | `required`  |
| elements       | Elements to animate.                                | `object`          | `undefined` |
| animation      | Animation to use.                                   | `ActionAnimation` | `undefined` |
| memory         | Memory to use.                                      | `object`          | `undefined` |
