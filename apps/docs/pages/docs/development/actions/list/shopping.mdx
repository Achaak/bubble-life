# Shopping actions

## addShoppingActionInWaitingList()

Add an shopping action in the await list.

```typescript
import { addShoppingActionInWaitingList } from "@bubble/core";

addShoppingActionInWaitingList({
  start: new Date().getTime(),
  duration: 10000,
  importance, 2
});
```

#### Parameters

| Parameter  | Description                  | Type        | Default    |
| ---------- | ---------------------------- | ----------- | ---------- |
| start      | The start time of the action | `number`    | `required` |
| duration   | The duration of the action   | `number`    | `required` |
| importance | The importance of the action | `1, 2 or 3` | `required` |

---

## addShoppingActionInWaitingListDefault()

Add an shopping action in the await list with default values.

```typescript
import { addShoppingActionInWaitingListDefault } from '@bubble/core'

addShoppingActionInWaitingListDefault()
```
