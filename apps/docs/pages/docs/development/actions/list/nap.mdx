# Nap actions

## addNapActionInAwaitList()

Add an nap action in the await list.

```typescript
import { addNapActionInAwaitList } from "@bubble/core";

addNapActionInAwaitList({
  start: new Date().getTime(),
  duration: 10000,
  importance, 2
});
```

#### Parameters

| Parameter  | Description                  | Type        | Default    |
| ---------- | ---------------------------- | ----------- | ---------- |
| start      | The start time of the action | `number`    | `required` |
| duration   | The duration of the action   | `number`    | `required` |
| importance | The importance of the action | `1, 2 or 3` | `required` |

---

## addNapActionInAwaitListDefault()

Add an nap action in the await list with default values.

```typescript
import { addNapActionInAwaitListDefault } from "@bubble/core";

addNapActionInAwaitListDefault();
```
