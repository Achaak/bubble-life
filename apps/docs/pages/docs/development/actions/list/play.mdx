# Play actions

## addPlayActionInWaitingList()

Add an play action in the await list.

```typescript
import { addPlayActionInWaitingList } from "@bubble/core";

addPlayActionInWaitingList({
  start: new Date().getTime(),
  duration: 10000,
  importance, 2
});
```

#### Parameters

| Parameter  | Description                  | Type        | Default    |
| ---------- | ---------------------------- | ----------- | ---------- |
| start      | The start time of the action | `number`    | `required` |
| duration   | The duration of the action   | `number`    | `required` |
| importance | The importance of the action | `1, 2 or 3` | `required` |

---

## addPlayActionInWaitingListDefault()

Add an play action in the await list with default values.

```typescript
import { addPlayActionInWaitingListDefault } from '@bubble/core'

addPlayActionInWaitingListDefault()
```
