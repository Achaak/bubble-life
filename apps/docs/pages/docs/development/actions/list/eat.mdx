# Eat actions

## addEatActionInWaitingList()

Add an eat action in the await list.

```typescript
import { addEatActionInWaitingList } from "@bubble/core";

addEatActionInWaitingList({
  start: new Date().getTime(),
  duration: 10000,
  importance, 2
});
```

#### Parameters

| Parameter  | Description                  | Type        | Default    |
| ---------- | ---------------------------- | ----------- | ---------- |
| start      | The start time of the action | `number`    | `required` |
| duration   | The duration of the action   | `number`    | `required` |
| importance | The importance of the action | `1, 2 or 3` | `required` |

---

## addEatActionInWaitingListDefault()

Add an eat action in the await list with default values.

```typescript
import { addEatActionInWaitingListDefault } from '@bubble/core'

addEatActionInWaitingListDefault()
```
