# Icons

## Sources

The project imports all the Material Icons from Google.

You can find and search them here : <a href="https://fonts.google.com/icons">Material Icons</a>

## How to use

### Types

```html
// Regular
<span className="material-icons">name_of_icon</span>

// Outlined
<span className="material-icons-outlined">name_of_icon</span>

// Round
<span className="material-icons-round">name_of_icon</span>

// Sharp
<span className="material-icons-sharp">name_of_icon</span>

// Two Tone
<span className="material-icons-two-one">name_of_icon</span>
```

### Responsive Sizes

```html
// All sizes
<span className="material-icons size-8">name_of_icon</span>

// Extra Small (480px)
<span className="material-icons size-xs-8">name_of_icon</span>

// Small (640px)
<span className="material-icons size-sm-8">name_of_icon</span>

// Medium (768px)
<span className="material-icons size-md-8">name_of_icon</span>

// Large (1024px)
<span className="material-icons size-lg-8">name_of_icon</span>

// Extra Large (1280px)
<span className="material-icons size-xl-8">name_of_icon</span>
```
