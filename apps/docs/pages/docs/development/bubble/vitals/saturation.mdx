# Saturation Vital

## setSaturation()

Set the saturation of the Bubble.

```typescript
import { setSaturation } from "@bubble/store";

setSaturation({
  value: 42,
});
```

#### Parameters

| Parameter | Description                  | Type     | Default    |
| --------- | ---------------------------- | -------- | ---------- |
| value     | The saturation of the Bubble | `number` | `required` |

---

## resetSaturation()

Reset the saturation of the Bubble.

```typescript
import { resetSaturation } from "@bubble/store";

resetSaturation();
```
