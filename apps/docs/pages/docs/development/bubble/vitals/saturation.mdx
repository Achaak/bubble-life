# Saturation Vital

## setSaturation()

Set the saturation of the Bubble.

### Use in app module

```typescript
import { setSaturation } from '@bubble/store';

setSaturation({
  value: 42,
});
```

### Use in controller module

```typescript
import { socket } from '@bubble/common';

socket().emit('setSaturation', {
  value: 42,
});
```

#### Parameters

| Parameter | Description                  | Type     | Default    |
| --------- | ---------------------------- | -------- | ---------- |
| value     | The saturation of the Bubble | `number` | `required` |
