

# Tiredness Vital

## setTiredness()

Set the tiredness of the Bubble.

```typescript
import { setTiredness } from "@bubble/store";

setTiredness({
  value: 42,
});
```

#### Parameters

| Parameter | Description                 | Type     | Default    |
| --------- | --------------------------- | -------- | ---------- |
| value     | The tiredness of the Bubble | `number` | `required` |

---

## resetTiredness()

Reset the tiredness of the Bubble.

```typescript
import { resetTiredness } from "@bubble/store";

resetTiredness();
```
