# Tiredness Vital

## setTiredness()

Set the tiredness of the Bubble.

### Use in app module

```typescript
import { setTiredness } from '@bubble/store';

setTiredness({
  value: 42,
});
```

### Use in controller module

```typescript
import { socket } from '@bubble/common';

socket().emit('setTiredness', {
  value: 42,
});
```

#### Parameters

| Parameter | Description                 | Type     | Default    |
| --------- | --------------------------- | -------- | ---------- |
| value     | The tiredness of the Bubble | `number` | `required` |
