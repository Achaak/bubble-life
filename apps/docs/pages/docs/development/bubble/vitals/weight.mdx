# Weight Vital

## setWeight()

Set the weight of the Bubble.

### Use in app module

```typescript
import { setWeight } from '@bubble/store'

setWeight({
  value: 42,
})
```

### Use in controller module

```typescript
import { socket } from '@bubble/common'

const socket = socket()
socket.emit('setWeight', {
  value: 42,
})
```

#### Parameters

| Parameter | Description              | Type     | Default    |
| --------- | ------------------------ | -------- | ---------- |
| value     | The weight of the Bubble | `number` | `required` |
