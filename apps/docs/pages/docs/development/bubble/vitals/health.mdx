# Health Vital

## setHealth()

Set the health of the Bubble.

### Use in app module

```typescript
import { setHealth } from '@bubble/store';

setHealth({
  value: 42,
});
```

### Use in controller module

```typescript
import { socket } from '@bubble/common';

socket().emit('setHealth', {
  value: 42,
});
```

#### Parameters

| Parameter | Description              | Type     | Default    |
| --------- | ------------------------ | -------- | ---------- |
| value     | The health of the Bubble | `number` | `required` |
