# Happiness Vital

## setHappiness

Set the happiness of the Bubble.

### Use in app module

```typescript
import { setHappiness } from '@bubble/store';

setHappiness({
  value: 42,
});
```

### Use in controller module

```typescript
import { socket } from '@bubble/common';

socket().emit('setHappiness', {
  value: 42,
});
```

#### Parameters

| Parameter | Description                 | Type     | Default    |
| --------- | --------------------------- | -------- | ---------- |
| value     | The happiness of the Bubble | `number` | `required` |
