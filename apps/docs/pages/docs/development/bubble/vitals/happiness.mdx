# Happiness Vital

## setHappiness

Set the happiness of the Bubble.

```typescript
import { setHappiness } from "@bubble/store";

setHappiness({
  value: 42,
});
```

#### Parameters

| Parameter | Description                 | Type     | Default    |
| --------- | --------------------------- | -------- | ---------- |
| value     | The happiness of the Bubble | `number` | `required` |

---

## resetHappiness

Reset the happiness of the Bubble.

```typescript
import { resetHappiness } from "@bubble/store";

resetHappiness();
```
