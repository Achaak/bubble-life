# Animations

## resetAnimation()

Reset the animation to the Bubble

### Use in app module

```typescript
import { resetAnimation } from '@bubble/store'

resetAnimation()
```

### Use in controller module

```typescript
import { socket } from '@bubble/common'

const socket = socket()
socket.emit('resetAnimation')
```

---

## addAnimationInList()

Set the tiredness of the Bubble.

### Use in app module

```typescript
import { addAnimationInList } from '@bubble/store'

addAnimationInList({
  id: 'id',
  importance: 1,
  name: 'AnimationName',
})
```

### Use in controller module

```typescript
import { socket } from '@bubble/common'

const socket = socket()
socket.emit('addAnimationInList', {
  id: 'id',
  importance: 1,
  name: 'AnimationName',
})
```

#### Parameters

| Parameter  | Description                     | Type        | Default    |
| ---------- | ------------------------------- | ----------- | ---------- |
| id         | The id of the animation         | `number`    | `required` |
| importance | The importance of the animation | `1, 2 or 3` | `required` |
| name       | The name of the animation       | `Animation` | `required` |
