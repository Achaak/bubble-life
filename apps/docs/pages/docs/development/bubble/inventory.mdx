# Inventory

## addInventoryItem()

Add an item to the inventory.

```typescript
import { addInventoryItem } from "@bubble/store";

addInventoryItem({
  type: "InventoryItem",
  number: 1,
});
```

#### Parameters

| Parameter | Description                 | Type            | Default    |
| --------- | --------------------------- | --------------- | ---------- |
| type      | The type of the item to add | `InventoryItem` | `required` |
| number    | The number of items to add  | `number`        | `required` |
