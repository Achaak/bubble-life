# Inventory

## addInventoryItem()

Add an item to the inventory.

### Use in app module

```typescript
import { addInventoryItem } from '@bubble/store'

addInventoryItem({
  type: 'InventoryItem',
  number: 1,
})
```

### Use in controller module

```typescript
import { socket } from '@bubble/common'

const socket = socket()
socket.emit('addInventoryItem', {
  type: 'InventoryItem',
  number: 1,
})
```

#### Parameters

| Parameter | Description                 | Type            | Default    |
| --------- | --------------------------- | --------------- | ---------- |
| type      | The type of the item to add | `InventoryItem` | `required` |
| number    | The number of items to add  | `number`        | `required` |
