# Bodies Element

## addBodyInList()

Add a body to the list of bodies.

### Use in app module

```typescript
import { addBodyInList } from '@bubble/store'

addBodyInList({
  id: 'bodyId',
  importance: 1,
  name: 'BodyName',
})
```

### Use in controller module

```typescript
import { socket } from '@bubble/common'

const socket = socket()
socket.emit('addBodyInList', {
  id: 'bodyId',
  importance: 1,
  name: 'BodyName',
})
```

#### Parameters

| Parameter  | Description                            | Type        | Default    |
| ---------- | -------------------------------------- | ----------- | ---------- |
| id         | Id of the body to add.                 | `string`    | `required` |
| importance | Importance of the action to check for. | `1, 2 or 3` | `required` |
| name       | Name of the action to check for.       | `Bodies`    | `required` |
