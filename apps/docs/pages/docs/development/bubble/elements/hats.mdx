# Hats Element

## addHatInList()

Add a hat to the list of hats.

```typescript
import { addHatInList } from "@bubble/store";

addHatInList({
  id: "hatId",
  importance: 1,
  name: "HatName",
});
```

#### Parameters

| Parameter  | Description                            | Type        | Default    |
| ---------- | -------------------------------------- | ----------- | ---------- |
| id         | Id of the hat to add.                  | `string`    | `required` |
| importance | Importance of the action to check for. | `1, 2 or 3` | `required` |
| name       | Name of the action to check for.       | `Hats`      | `required` |

---

## removeHatInList()

Remove a hat from the list of hats.

```typescript
import { removeHatInList } from "@bubble/store";

removeHatInList({
  id: "hatId",
});
```

#### Parameters

| Parameter | Description              | Type     | Default    |
| --------- | ------------------------ | -------- | ---------- |
| id        | Id of the hat to remove. | `string` | `required` |

---

## resetHat()

Reset the hat state.

```typescript
import { resetHat } from "@bubble/store";

resetHat();
```

---

## setActionHat()

<Callout emoji="⚠️">
  This function is only usable in the core package with the actions class
  manager.
</Callout>

Set the action hat.

```typescript
import { setActionHat } from "@bubble/store";

setActionHat({
  actionId: "actionId",
  name: "hatName",
});
```

#### Parameters

| Parameter | Description                | Type     | Default    |
| --------- | -------------------------- | -------- | ---------- |
| actionId  | Id of the action to set.   | `string` | `required` |
| name      | Name of the action to set. | `Hats`   | `required` |

---

## resetActionHat()

Reset the action hat.

```typescript
import { resetActionHat } from "@bubble/store";

resetActionHat();
```
