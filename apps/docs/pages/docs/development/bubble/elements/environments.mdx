# Environments Element

## addEnvironmentInList()

Add a environment to the list of environments.

```typescript
import { addEnvironmentInList } from "@bubble/store";

addEnvironmentInList({
  id: "environmentId",
  importance: 1,
  name: "EnvironmentName",
});
```

#### Parameters

| Parameter  | Description                            | Type           | Default    |
| ---------- | -------------------------------------- | -------------- | ---------- |
| id         | Id of the environment to add.          | `string`       | `required` |
| importance | Importance of the action to check for. | `1, 2 or 3`    | `required` |
| name       | Name of the action to check for.       | `Environments` | `required` |

---

## removeEnvironmentInList()

Remove a environment from the list of environments.

```typescript
import { removeEnvironmentInList } from "@bubble/store";

removeEnvironmentInList({
  id: "environmentId",
});
```

#### Parameters

| Parameter | Description                      | Type     | Default    |
| --------- | -------------------------------- | -------- | ---------- |
| id        | Id of the environment to remove. | `string` | `required` |

---

## resetEnvironment()

Reset the environment state.

```typescript
import { resetEnvironment } from "@bubble/store";

resetEnvironment();
```

---

## setActionEnvironment()

<Callout emoji="⚠️">
  This function is only usable in the core package with the actions class
  manager.
</Callout>

Set the action environment.

```typescript
import { setActionEnvironment } from "@bubble/store";

setActionEnvironment({
  actionId: "actionId",
  name: "environmentName",
});
```

#### Parameters

| Parameter | Description                | Type           | Default    |
| --------- | -------------------------- | -------------- | ---------- |
| actionId  | Id of the action to set.   | `string`       | `required` |
| name      | Name of the action to set. | `Environments` | `required` |

---

## resetActionEnvironment()

Reset the action environment.

```typescript
import { resetActionEnvironment } from "@bubble/store";

resetActionEnvironment();
```
