# Global Configuration

## Configure

1. Copy `/packages/configs/global.default.ts` to `/packages/configs/global.ts`.

2. Modify `/packages/configs/global.ts` to your needs.

---

## Properties

### Global

Global properties are used to configure the global behavior of the application.

| Option      | Description                           |
| ----------- | ------------------------------------- |
| `lang`      | The language of the application.      |
| `clickable` | Whether the application is clickable. |

### Development

Development properties are used to configure the development behavior of the application.

| Option     | Description                                   |
| ---------- | --------------------------------------------- |
| `showFPS`  | Whether to show the FPS.                      |
| `devTools` | Whether to show the dev tools.                |
| `kiosk`    | Whether to run the application in kiosk mode. |
| `menuBar`  | Whether to show the menu bar.                 |
