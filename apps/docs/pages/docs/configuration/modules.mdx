# Modules Configuration

## Configure

1. Copy `/packages/modules/configs/src/modules.default.ts` to `/packages/modules/configs/src/modules.ts`.

2. Modify `/packages/modules/configs/src/modules.ts` to your needs.

---

## Properties

When you want to use a module, you need to add a property to `/packages/modules/configs/src/modules.ts`.

You can check how to install a module in this <a href="/modules/installation">page</a>.

| Option     | Description                                            |
| ---------- | ------------------------------------------------------ |
| `module`   | You need import the module and add it to the property. |
| `position` | The position of the module.                            |
| `config`   | The configuration of the module.                       |

#### Positions

- `top-left`
- `top`
- `top-right`
- `center-left`
- `center`
- `center-right`
- `bottom-left`
- `bottom`
- `bottom-right`
